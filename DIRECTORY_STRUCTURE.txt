KEY NOTES:
═════════

1. .gitignore files:
   - data/config.json           (API keys, secrets)
   - data/trades.db             (Database - regenerable)
   - data/historical/           (Downloaded data - regenerable)
   - logs/                      (Log files)
   - __pycache__/               (Python cache)
   - venv/                      (Virtual environment)

2. Create these directories manually:
   mkdir -p src/{data,analysis,signals,backtest,learning,risk,broker,utils}
   mkdir -p tests/fixtures
   mkdir -p notebooks
   mkdir -p docs
   mkdir -p data/{historical,backtest_results}
   mkdir -p logs

3. Phase 1 will focus on:
   - src/analysis/              (Core PA detection)
   - src/signals/               (Entry signals)
   - tests/                     (Unit tests)

4. Add as needed:
   - docs/API_REFERENCE.md      (After Phase 1)
   - .github/workflows/         (After Phase 2, for CI/CD)
   - notebooks/                 (After Phase 3, for analysis)

5. Python path:
   - All imports use: from src.module import Component
   - Or add to sys.path in main.py
